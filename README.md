# Financial Modeling Review Samples

This repository contains structured financial modeling samples prepared as part of project finance, projections, and valuation workflows.

The objective of these samples is to demonstrate practical experience in reviewing financial assumptions, validating model outputs, and ensuring internal consistency across financial statements.

Key areas covered:

• Revenue projection validation  
• Cost structure alignment  
• Cash flow consistency checks  
• Balance sheet linkage logic  
• Means of finance structure review  
• DSCR and coverage ratio analysis  
• Assumption testing against industry benchmarks  

These examples reflect my experience working with structured financial datasets in treasury and reporting environments at ICICI Bank.

I am particularly interested in contributing to AI-assisted financial reasoning evaluation workflows involving validation of projections, DCF logic, and model consistency.
